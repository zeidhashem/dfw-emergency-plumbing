# DFW Emergency Roadside (DFW) — Ads-Optimized Landing Page

Single-page static site designed for Google Ads traffic:
- Call-first layout (phone everywhere)
- Mobile sticky call bar
- Fast load, minimal scripts
- LocalBusiness schema

## Add the background image
Place your hero image here:
`assets/roadside-bg.jpg`

Recommended: JPG, 1600–2400px wide, compressed.

## Deploy on GitHub Pages
1. Create repo
2. Add files
3. Settings → Pages → Deploy from branch (`main`) / root
4. Optional: connect your domain

## Google Ads tracking (optional)
Paste your Google tag snippet in `index.html` head.
Then update the conversion send_to line in `script.js`.
